Contains simple arithmetic components such as basic math operators (+-*/) and math functions.

Contains comparison operators such as, but not limited to, Greater Than, Less Than and Equal To.

See Comparison reference

Logic

Contains logical operators such as, but not limited to, And, Or and Not.

See Logic reference

I/O (Input/Output)

Contains basic user input components for keyboard and mouse, such as a Button component.

See I/O reference

Bitwise Operations

Contains operators to work with binary. All numeric values are floored.

See Bitwise Operations reference

Storage (Memory)

Contains components to hold onto values for later use.

See Storage (Memory) reference

Time

Contains components that measure time or use time for calculations.

See Time reference

Trigonometry

Contains trigonometry functions such as Sine, Cosine and Tangent.

See Trigonometry reference

Graphics

Contains graphical components to visualize values.
