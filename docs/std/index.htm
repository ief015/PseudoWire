What is the Standard Lua Package?

The Standard Lua Package (SLP) is the addon that is included with Pseudo Wire.

The Standard Lua Package provides only basic components for use in vanilla (default) Pseudo Wire. The SLP contains lots of minimalistic componented under nine basic categories;

Math
Comparison
Logic
I/O (Input/Output)
Bitwise Operations
Storage (Memory)
Time
Trigonometry
Graphics

An explaination of each category can be found here.
