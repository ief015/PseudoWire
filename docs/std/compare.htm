Greater Than

Greater Than. If A > B then Out is 1, else Out is 0.

Inputs

A Input A.
B Input B.

Outputs

Out True if A > B.

Less Than

Less Than. If A < B then Out is 1, else Out is 0.

Inputs

A Input A.
B Input B.

Outputs

Out True if A < B.

Equal To

Equal To. If A = B then Out is 1, else Out is 0.

Inputs

A Input A.
B Input B.

Outputs

Out True if A = B.

Greater Than / Equal

Greater Than or Equal To. If A >= B then Out is 1, else Out is 0.

Inputs

A Input A.
B Input B.

Outputs

Out True if A >= B.

Less Than / Equal

Less Than or Equal To. If A <= B then Out is 1, else Out is 0.

Inputs

A Input A.
B Input B.

Outputs

Out True if A <= B.

Not Equal To

Not Equal To. If NOT A = B then Out is 1, else Out is 0. (Opposite of Equal To)

Inputs

A Input A.
B Input B.

Outputs

Out True if NOT A = B.
