Pseudo Wire Reference

Intro

What is Pseudo Wire?

Pseudo Wire is a scriptable logic simulator designed as a recreation of Wiremod (an addon for Garry's Mod) in a standalone environment.
Create logical machines. Spawn parts and components, wire them together, and interact with them.

Pseudo Wire is designed so that anyone who has used Wiremod before would be able to pick up how to play in almost no time at all. Any newcomers shouldn't take too long get the hang of how the game plays, feels and works.

Requirements

A half-decent computer with an OpenGL compliant video card.
That's it.

Note: Some advanced graphics components (more specifically, components that use RenderTextures) may have an unstable effect on Intel Integrated Graphics.

Credits

Pseudo Wire was written in C++ by Nathan Cousins.
Pseudo Wire uses Lua for scripting, SFML for OpenGL graphics handling and TinyXML for XML parsing.
