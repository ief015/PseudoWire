1. Menu Bar

The menu bar is located at the top of the screen.

The menu bar has eight items that can be selected:

  • New
  • Save
  • Load
  • Edit / Game Mode
  • Show Grid / Remove Grid
  • Enable / Disable Battery Saver
  • About
  • Exit

New clears the game of all components, leaving the user a clean slate to build a new creation.

Save allows the user to save their workspace to a file.

Load allows the user to open saved workspaces and continue working on them or interact with them.

Edit / Game Mode is a toggleable item on the Menu bar. During Edit Mode, the user can build and edit their components. Edit Mode displays an icon beside the mouse to ensure the user knows they are in Edit Mode. During Game Mode, the user has full interaction with the components. All tools are disabled in Game Mode.

Show Grid / Remove Grid is a toggleable item in the Menu Bar. When enabled, a "snap-to" grid will appear on the game workspace. Use the accompanying Up and Down buttons can grow or shrink the grid size.

Enable / Disable Battery Saver is a toggleable item in the Menu Bar. When enabled, the game will give slack to the CPU, allowing a longer battery life on laptops. However, this will drop performance.

About displays Pseudo Wire information, such as Authors and Version.

Exit will quit the game. Any unsaved progress will be lost.

2. Spawn List

The spawn list (or spawn menu) is a tree of all the different components you can build and use.
You can access this menu by Right-Clicking on an empty space in the game during Edit Mode.
When you first open the menu, you'll see all the different categories you can expand.
Categories are used to group like-components. Categories are items with the or icons beside them.
Use the scroll wheel or arrow keys to scroll up and down the list.

For example, if you click on the Math category, you'll see math-related components, such as Add, Subtact, Absolute, Round, Negate, and so on.
You may click on a component to spawn it.

3. Context Menu

The context menu contains all component editing tools.
You can access this menu by Right-Clicking on the component you wish to edit during Edit Mode.

When you first open the context menu, you have many different options:

  • Wire Input
  • Wire Output
  • Edit
  • Unwire Input
  • Unwire Output
  • Remove

Wire Input is used to connect (or "wire") an input of this component to an output of another component. Choosing this will open another menu with all the inputs of the component you wish to connect to. After selecting an input, right click on the other component you want to connect to, and click on an available output.

Wire Output is much like the above. It is used to connect (or "wire") an output of this component to an input of another component. If you have played with Wiremod, this will probably be your prefered method of wiring.

Edit will bring up a dialog window that may give you options to edit how the component functions. However, not all components are customizable and choosing this will perform no action.

Unwire Input is used to remove the connection on an input. Clicking this will bring up a list of inputs to choose from.

Unwire Output is used to remove all connections on an output. Clicking this will bring up a list of outputs to choose from.

Remove is used to delete a component from the game. Clicking this will first remove any connections, then delete the component.

4. Save Window

The saving window allows you to save your progress and load it another time.
Enter your filename into the textbox (the .pwm extension is not required) and hit the Accept button at the top right (green.) This will create a .pwm (Pseudo Wire Machine) save file which will be stored in your saves folder.
To cancel the saving window, hit the Cancel button at the top right (red.)
If "Save output values" is checked, the game will save the values of all component outputs and will be reloaded in game load. This feature is rarely used and not needed. This will also increase the save file's size.

5. Load Window

The loading window allows you to load progress that was previously saved.
Click on the file that you wish to load and click the Accept button at the top right (green.)
To cancel the loading window, hit the Cancel button at the top right (red.)
You may use the scroll wheel or arrow keys to scroll up and down.
The loading window explores the saves folder, where all save files should be located.

6. Edit Window

The edit window allows you to modify the component and how it behaves. Not all components will have this feature.
To submit new settings, hit the Accept button (green.)
To cancel new settings, hit the Cancel button (red.)
