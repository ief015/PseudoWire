What are Addons/Packages?

Addons, also known as Packages, are a package of Lua scripts, images, and other resources bundled up into a format that Pseudo Wire can use.
For the sake of this reference, "package" will refer to the root addon folder and its contents and "addon" will refer to the addon as a whole.

An example of a package is the Standard Lua Package, which can be found in the addons folder with Pseudo Wire, called standard.

Addons usually consist of three folders: lua, img, fnt and an XML file, info.xml.
The lua folder contains all of the Lua scripts the package is associated with.
The img folder constains all of the image files that the package will use.
The fnt folder constains all of the font files that the package will use.

Not all packages will contain all folders, as some packages may associate their Lua scripts with other addon images.
However, all addons must contain the info.xml file, which is read at the root directory of the package so Pseudo Wire knows what Lua scripts to include, their names, categories, and other properties.

See the Create Addon page for more information on the info.xml file.
