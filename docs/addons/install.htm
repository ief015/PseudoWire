Addon Installation

Addon installation couldn't be any easier.

Simply place the root addon folder into the ../PseudoWire/addons/ folder. And, you're done.
The root addon folder can be determined by opening it, and seeing if there is a info.xml file. If so, that is the folder that should be placed into the ../PseudoWire/addons/ folder.
If an addon is placed into the ../PseudoWire/addons/ folder without an info.xml file, the addon will not be included and skipped.
