The lua Folder

../PseudoWire/addons/ExampleAddon/lua/

The lua folder contains all the Lua scripts the addon is associated with.
Inside the lua folder contains folders that represent categories in the Spawn List in-game.
All respective scripts are located in these category folders.

Example: ../PseudoWire/addons/ExampleAddon/lua/math/addition.lua

The img Folder

../PseudoWire/addons/ExampleAddon/img/

The img folder contains all image resources to be included with the package.

Example: ../PseudoWire/addons/ExampleAddon/img/addingchip.png

The fnt Folder

../PseudoWire/addons/ExampleAddon/fnt/

The fnt folder contains all font resources to be included with the package.

Example: ../PseudoWire/addons/ExampleAddon/fnt/myfont.ttf

info.xml

../PseudoWire/addons/ExampleAddon/info.xml

The info.xml file is used to send information about the package to Pseudo Wire.

info.xml include information about the package itself; Name, Author and Version.
It also carries information about the Lua scripts associated with the package. This information includes; Name, Category, Path Name, and a short description.

Note: info.xml must be placed in the root of the addon folder, or the package will be ignored and skipped!

Here's a small example of an info.xml file:

<package name="Example" author="Jamie" version="0.9.0 BETA">
  <category name="Math" path="math">
    <item name="Add" path="addition.lua"> Addition. (Out = A + B + C + D) </item>
  </category>
</package>

<package>

name: A string of text that represents the name of the package. (This and the addon folder name do not need to be the same.)
author: A string of text that represents the name of the author(s) who wrote the package.
version: A string of text that represents the version of the package.

<category>

name: The name of the category displayed in-game.
path: The folder name inside ../PseudoWire/addons/Example/lua/

<item>

name: The name of the component displayed in-game.
path: The Lua script file inside the respective category folder. (see <category> above)
Value between tags: A short description of the item displayed in-game.
